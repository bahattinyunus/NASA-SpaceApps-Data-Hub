# Contributing to NASA SpaceApps Data Hub

First off, thank you for considering contributing to this NASA SpaceApps project! It's people like you that make the world's open science data accessible and impactful.

## How Can I Contribute?

### Reporting Bugs
* Check the existing issues to ensure the bug hasn't been reported yet.
* Open a new issue with a clear title and description, including steps to reproduce the bug.

### Suggesting Enhancements
* Open an issue with the "enhancement" label.
* Describe the feature you'd like to see and why it would be useful for Earth observation analysis.

### Pull Requests
1. Fork the repo and create your branch from `main`.
2. If you've added code that should be tested, add tests.
3. Ensure the test suite passes.
4. Make sure your code lints.
5. Issue that pull request!

## Styleguide
* Use PEP 8 for Python code.
* Document new functions and classes with descriptive docstrings.
* Use clear, descriptive commit messages.

## Community
* Join our Space Apps community discussions.
* Be respectful and collaborative.

Together, we can build a better bridge to our planet's future!
